REST API 设计

FastAPI 后端 — V1 MVP 端点

认证

方法 路径 说明
POST/api/auth/register注册（email + password）
POST/api/auth/login登录 → 返回 JWT

上传 & 解析

方法 路径 说明
POST/api/upload上传交割单文件 → 保存 RawFile → 格式检测 → 返回候选格式列表
POST/api/upload/confirm用户确认格式 → 执行解析 → 返回 Trade 预览列表
POST/api/upload/import确认无误后导入 → Trade[] 入库 → 返回导入摘要

分析（核心）

方法 路径 说明
POST/api/analysis/run对指定日期范围的 Trade 执行完整分析管道（Position → Pattern → Insight → What If），返回完整结果
GET/api/analysis/{analysis_id}/stats查看某次分析的交易统计结果
GET/api/analysis/{analysis_id}/insight查看 Insight 归因结果（各 Pattern 胜率/收益排名）
GET/api/analysis/{analysis_id}/whatif查看 What If 回测结果（删除每种行为后的收益变化）

AI 报告

方法 路径 说明
POST/api/report/generate传入 analysis_id → AI 生成诊断报告 → 验证层校验 → 返回报告
GET/api/report/{report_id}查看已生成的报告
GET/api/reports用户的历史报告列表

上传 → 报告的完整流程

前端调用顺序
① POST /api/upload → 返回 detected_formats
② POST /api/upload/confirm {source_type} → 返回 Trade[] 预览
用户在前端查看预览表格，确认数据正确
③ POST /api/upload/import → Trade[] 入库
④ POST /api/analysis/run {date_range} → 后台执行管道，返回 analysis_id
⑤ GET /api/analysis/{id}/stats → 渲染统计面板
⑥ GET /api/analysis/{id}/insight → 渲染行为归因面板
⑦ GET /api/analysis/{id}/whatif → 渲染 What If 回测面板
⑧ POST /api/report/generate → AI 生成报告 → 渲染报告页
